## ğŸš€ Setup Instructions

### Prerequisites
- A web browser (Chrome, Firefox, Safari, etc.)
- Text editor (VS Code recommended)
- Local server (optional, for development)

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/Shrutisharma7411/portfolio.git
   cd portfolio
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - For a better development experience, use a local server

3. **Using Python (recommended)**
   ```bash
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000`

4. **Using Node.js**
   ```bash
   npx serve .
   ```

## ğŸ¨ Customization

### Colors
Modify the CSS variables in `css/style.css`:
```css
:root {
  --primary-color: #ffcd42;    /* Yellow accent */
  --secondary-color: #ffd35c;  /* Light yellow */
  --bg-primary: #ffffff;       /* White background */
  --text-color: #222222;       /* Dark text */
}
```

### Content
Update the following sections in `index.html`:
- Personal information in the hero section
- Project details and links
- Education and experience entries
- Skills list
- Contact information

### Images
Replace images in the `assets/` folder:
- `assets/data/pic.png` - Your profile picture
- `assets/data/ShrutiCV.pdf` - Your resume
- `assets/project/` - Project screenshots
